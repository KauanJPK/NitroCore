# Changelog

All notable changes to this project will be documented in this file.

This project follows Semantic Versioning.

---

## [0.0.2] - Foundation

### Added

- Introduced the NitroCore Optimization Framework.
- Added `OptimizationManager`.
- Added `OptimizationRegistry`.
- Added `OptimizationModule`.
- Added `OptimizationType`.
- Added `OptimizationProfile`.
- Added the first optimization module: `ParticleOptimization`.
- Improved internal project architecture.
- Prepared the engine for future rendering optimizations.

### Changed

- Improved project organization.
- Refactored initialization flow.

---

## [0.0.1] - Genesis

### Added

- Initial project setup.
- GitHub repository.
- GTNH Convention Plugin.
- UniMixins support.
- Centralized logging system (`NitroLog`).
- Initial documentation.
- Branding assets.
